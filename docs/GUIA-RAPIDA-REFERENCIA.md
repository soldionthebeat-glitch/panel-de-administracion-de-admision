## 📊 FÓRMULA RÁPIDA DE PUNTAJE

### Calcula en 30 segundos:

**PUNTAJE FINAL = (A×0.25) + (T×0.20) + (H×0.20) + (O×0.20) + (C×0.15)**

Donde:
- **A** = Promedio Artístico (1-5)
- **T** = Promedio Técnico (1-5)
- **H** = Promedio Humano (1-5)
- **O** = Promedio Organizativo (1-5)
- **C** = Promedio Comercial (1-5)

### Ejemplo:
```
A = 4.0 × 0.25 = 1.00
T = 3.5 × 0.20 = 0.70
H = 4.2 × 0.20 = 0.84
O = 3.8 × 0.20 = 0.76
C = 3.0 × 0.15 = 0.45
              ------
TOTAL = 3.75 → MODELO B/D
```
