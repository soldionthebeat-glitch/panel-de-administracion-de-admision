## 🎯 Puntuación Rápida de Módulos

```

ARTÍSTICA (25%)
  + Géneros detallados (1.5)
  + Referencias claras (1.5)
  + Sonido definido (1.5)
  + Diferenciación (2.0) ← CRÍTICO
  + Producciones (1.5-2)
  + Enfoque claro (1)
  + Versatilidad (1.5)
  ──────────────────────
  Total = Promedio × 2 = 0-10

TÉCNICA (20%)
  + Experiencia (1.5-9)
  + DAW (2)
  + Stems (0-3)
  + BPM/Tonalidad (0-3)
  + Organización (1.5-2.5)
  + Mezcla (0-3)
  + Mastering (0-3)
  + Samples/IA (0-3)
  + Licencias (0-2)
  ──────────────────────
  Total = Promedio × 1.2 = 0-10

[SIMILAR PARA HUMANO, ORGANIZATIVO, COMERCIAL]
```
