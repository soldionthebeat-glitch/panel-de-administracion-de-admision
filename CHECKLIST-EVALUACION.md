## 🚀 COMIENZA AQUÍ

**Cuando recibas una solicitud, sigue este checklist en orden.**

**Tiempo estimado:** 15-30 minutos por solicitud

---

## PASO 1: VALIDACIÓN INICIAL (2 minutos)

- [ ] **¿Formulario completo?**
  - Todos los campos requeridos completados
  - Sin errores obvios de tipeo
  - Email verificable
  
  **SI NO:** Contactar productor pidiendo completar
  
- [ ] **¿Mayor de 18 años?**
  - Edad ≥ 18
  
  **SI NO:** RECHAZO AUTOMÁTICO

- [ ] **¿Email y contacto válidos?**
  - Email con @ y dominio
  - WhatsApp o teléfono verificable
  
  **SI NO:** Contactar para datos correctos

## PASO 2: FILTROS CRÍTICOS (3 minutos)

### ❌ PREGUNTAS QUE CAUSAN RECHAZO AUTOMÁTICO

**Ir directamente al módulo 7 (Legal) y revisar:**

- [ ] **¿Conflictos de derechos?**

  → Ir a: "Comunicación de Rechazo"

  ```
  Cualquier declaración sin ✓
  ```

## PASO 3: REVISIÓN LEGAL DETALLADA (5 minutos)

**Si pasó filtros críticos, revisar módulo 7 más profundamente:**

- [ ] **Contratos vigentes**
  - SÍ → Solicitar documentación
  - Revisar qué derechos cedió
  - **Acción:** Añadir nota "REVISAR CONTRATO"
  
- [ ] **Uso de samples/loops**
  - SÍ + Con licencias → BIEN
  - SÍ + Sin licencia → RECHAZO (ya capturado)
  - SÍ + IA → Solicitar especificación
  - **Acción:** Documentar tipo de material

- [ ] **Comparte derechos**
  - SÍ → Solicitar documentación de splits
  - Verificar acuerdos previos
  - **Acción:** Solicitar prueba de acuerdos

- [ ] **Registro en sociedades gestión**
  - SADAIC, DNDA, BMI, ASCAP, SESAC
  - **Acción:** Documentar si aplica

## PASO 9: CALCULAR PUNTAJE FINAL (1 minuto)

**Usar la fórmula:**

```
PUNTAJE FINAL = 
  (Artístico × 0.25) +
  (Técnico × 0.20) +
  (Humano × 0.20) +
  (Organizativo × 0.20) +
  (Comercial × 0.15)
```

**Reemplazar valores:**

```
PUNTAJE FINAL = 
  (_____ × 0.25) +
  (_____ × 0.20) +
  (_____ × 0.20) +
  (_____ × 0.20) +
  (_____ × 0.15)
  
  = _____ + _____ + _____ + _____ + _____
  
  = PUNTAJE FINAL: _____/5.0
```

**Convertir a escala 100:**

```
ESCALA 100 = (PUNTAJE FINAL / 5) × 100
           = (_____ / 5) × 100
           = _____/100
```

## ⚠️ SITUACIONES ESPECIALES

### ¿Qué hacer si...?

**... tiene un aspecto muy débil pero otros muy fuertes?**
```
→ Considerar Modelo A/D
→ Ofertar formación enfocada
→ Reevaluación en 3-6 meses
```

**... no tengo claridad en algo?**
```
→ Contactar directamente
→ Solicitar aclaración
→ Obtener documentación
→ No dejar dudas sin resolver
```

**... tiene información conflictiva?**
```
→ Revisar dos veces
→ Si conflicto legal: RECHAZO
→ Si conflicto menor: Aclarar
```

**... está entre dos clasificaciones?**
```
→ Usar decimal en puntaje (ej: 3.7)
→ Ir hacia la clasificación más baja para ser conservador
→ Si duda: Consultar con supervisor
```
