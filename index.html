🚀 Setup para Producción

Paso 1: Crear Cuenta Neon

Ve a https://neon.tech y crea una cuenta

Paso 2: Crear Proyecto

En Neon Dashboard: Create Project → nombre: producers_go

Paso 3: Crear Tablas

En SQL Editor: Copia y pega database.sql → Execute

Paso 4: Configurar Backend

cp .env.example .env # Edita .env con tu DATABASE_URL de Neon npm install npm start # Espera: "Server running on port 5000"

Paso 5: Test

curl http://localhost:5000/api/test # Debes ver: {"status":"OK",...}

Paso 6: Deploy (Render)

Push a GitHub → Conecta con Render → Deploy automático
